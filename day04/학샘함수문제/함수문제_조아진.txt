
         1.  홀수인지판단하는 함수 만들기  ( 수 -> 홀수이면  true, false  반환 )

         
            📝
            function 홀수(number) {
            
                if (number % 2 !== 0) {
                    return true;  
                } else {
                    return false; 
                }
            }

            // --- 실행 테스트 ---
            console.log(홀수(3)); // 결과: true
            console.log(홀수(4)); // 결과: false

         2.  성인판단하는 함수 만들기  (  나이  -> 20세 이상이면 true, false 반환 )


            📝
            function 성인(age) {
                if (age >= 20) {
                    return true;
                } else {
                    return false;
                }
            }

            // --- 실행 테스트 ---
            console.log(성인(25)); // 결과: true
            console.log(성인(15)); // 결과: false

         5.  두 수를 매개변수로 받아서  합을 구하는 함수 만들기 ( 두 수  - > 합을 반환하기 )

                📝
                    function 두수의합(num1, num2) {
                        return num1 + num2; 
                    }

                    // --- 실행 테스트 ---
                    console.log(두수의합(10, 20)); // 결과: 30
